##HTML 기본구조

'''
<!DOCTYPE html>
<html>
    <head>
        <title>HRML 문서의 제목입니다. </title>
    </head>
    <body>
        <h1>가장 큰 제목크기 입니다.</h1>
        <p>단락을 적는 부분</p>
    </body>
</html>
'''

<!DOCTYPE html> : 현재문서가 HTML5 문서라는것을 알려준다.

<title> : 문서의 제목을 정의하며 웹문서 툴바에 표시된다.

<body> : 웹 브라우저를 통해 보이는 내용 부분.

<h1> ~ <h6> : 제목을 나타낸다 h1이 가장크고 h6이 가장 작다.

<p> : 단락을 나타냅니다.
